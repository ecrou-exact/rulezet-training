# Rulezet Training

A centralized collection of security and logic rules for various platforms. This repository is designed as a resource library for workshops, CTFs, and technical training sessions, providing ready-to-import rules in multiple formats.

## formats

- YARA [doc](https://virustotal.github.io/yara/) **4 rules** and **1 bad rule**
- SIGMA [doc](https://sigmahq.io/docs/basics/rules.html) **3 rules** and **1 bad rule**
- SURICATA [doc](https://docs.suricata.io/en/latest/rules/index.html) **3 rules**
- CRS [doc](https://coreruleset.org/docs/3-about-rules/creating/) **3 rules**
- ZEEK [doc](https://docs.zeek.org/en/lts/scripting/basics.html)
- WAZUH [doc](https://documentation.wazuh.com/current/user-manual/ruleset/ruleset-xml-syntax/rules.html) **7 rules**
- NOVA [doc](https://novahunting.ai/) **8 rules**
- NSE [doc](https://nmap.org/book/nse.html) **2 rules**

## Open Source and Attribution

The rules found in this repository are gathered from various official open-source projects and community repositories on GitHub.

- This project is not claiming ownership of these rules.

- The intent is to provide a curated collection for training, not to copy or plagiarize original work.

- All rules include references or headers pointing to their original authors and licenses (such as MIT, Apache 2.0, or GPL) whenever possible.

- We encourage users to visit the original repositories listed in our documentation to support the creators.
